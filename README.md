# 实训报告生成器

一个本地运行的实训报告生成工具。支持在网页中拖入多个 TXT 笔记，按文件名中的“月-日”自动合并同一天内容，再基于 Word 模板生成实训报告。

## 功能

- 拖拽上传 TXT 文件
- 按 `月-日-次` 文件名分组，例如 `7-4-01.txt` 和 `7-4-02.txt` 会合并为同一天
- 自动生成课程目标、课程内容、课程内容详情、课程代码及执行过程
- 保留 Word 模板第一页，正文按模板要求设置字体和行距
- 可选 OpenAI 兼容 API 进行 AI 增强生成
- 生成后可直接在网页下载 `.docx`

## 运行方式

1. 安装依赖：

```bash
pip install -r requirements.txt
```

2. 将实训报告模板放到脚本同目录，命名为：

```text
实训报告（过程）模板.docx
```

3. 双击：

```text
启动实训报告网页.bat
```

或手动运行：

```bash
python 实训报告生成器网页.py
```

4. 浏览器访问：

```text
http://127.0.0.1:8765/
```

## 主要文件

- `实训报告生成器.py`：核心 Word 生成逻辑
- `实训报告生成器网页.py`：本地网页服务
- `实训报告生成器.html`：网页 UI
- `启动实训报告网页.bat`：Windows 双击启动脚本

## 说明

AI 增强是可选功能。不填写 AI API 配置时，工具会完全使用本地规则生成报告。
